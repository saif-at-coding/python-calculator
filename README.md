# Python Calculator v2.1

A beginner-friendly command-line calculator written in Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Square
- Cube
- Square Root
- Cube Root
- Power
- Percentage
- Absolute Value
- Random Integer Generator
- Sine
- Cosine
- Tangent
- Inverse Sine
- Inverse Cosine
- Inverse Tangent
- Calculation History
- Recent History
- Input Validation
- Exception Handling

## Requirements

Python 3.11+

## Run

python Calculator v2.1\main.py

## Test

python Calculator v2.1\tests.py
