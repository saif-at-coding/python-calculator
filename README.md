# python-calculator

A beginner-friendly command-line calculator written in Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Square
- Cube
- Square Root
- Cube Root
- Power
- Percentage
- Absolute Value
- Random Integer Generator
- Calculation History
- Recent History
- Input Validation
- Exception Handling

## Requirements

Python 3.11+

## Run

python Calculator v2.0\main.py

## Project Structure

- main.py
- operations.py
- utils.py
- tests.py
